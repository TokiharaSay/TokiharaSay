AMI或Insyde Insyde 两种类型BIOS 解锁CFG LOCK以及修改DVMT

技术更新换代快，温馨提示，请注意文章的更新时间。
本文最后更新于：2021年5月6日 上午

原因

解锁CFG LOCK / DVMT 许多人都不会，并且网上文章一般都是以AMI的BIOS多一些，

Insyde Insyde BIOS 文章还是太少了，这次我整理了一下网络上的教程做一个总结。

Insyde BIOS 解锁CFG Lock 修改DVMT

首先Insyde BIOS 文件一般是xxx.fd，我建议是（使用Windows，因为Mac我不知道怎么提取）提取本机的BIOS文件来进行查看和修改

需要使用Intel CSME System Tools 来自一个俄文的一个论坛来进行提取BIOS文件

由于主板的不同，工具版本可能有所不同，我用的是 v11 r37，然后解压Intel CSME System Tools，

打开到Intel CSME System Tools v11 r18》Flash Programming Tool》WIN64目录

新建一个BIOS备份批处理脚本，新建一个TXT文本名字是backup.txt ，用记事本打开，粘贴下面备份的命令，保存并将TXT后缀更为.bat

cd /d %~dp0
fptw64.exe -bios -f change.fd

运行backup.bat会生成一个名为backup.fd的文件这个就是BIOS文件

然后使用 UEFITool （Win和MAC这工具差不多的）选File然后选Open image file打开backup.fd

然后Ctrl+F搜索，选择Text
